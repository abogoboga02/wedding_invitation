# Homepage Wireframe

## Section Order
1. Sticky navbar
2. Hero
3. Live preview highlight
4. Template showcase
5. Why choose us
6. Cara kerja
7. Pricing
8. CTA closing
9. Footer

## Desktop Behavior
- navbar: menu kiri, brand tengah, menu kanan + CTA
- hero: centered copy, two CTA buttons
- preview area: split layout hero card + supporting cards
- template section: editorial masonry feel
- pricing: 3 card comparison with one featured plan

## Mobile Behavior
- brand centered, hamburger di kiri
- hero tetap center dengan copy pendek
- preview card stack vertikal
- template cards jadi single-column
- pricing cards stack penuh

## Visual Hierarchy
- headline hero jadi fokus utama
- preview URL personal jadi proof kedua
- template showcase jadi proof visual ketiga
- pricing dekat bagian bawah untuk conversion

## Spacing
- section major: `py-16` mobile, `py-24` desktop
- card gap: `gap-4` sampai `gap-6`
- hero max width: sekitar `max-w-4xl`
- panel radius besar untuk feel premium dan airy

## CTA
- primary hero: `Buat Undangan Pertama`
- secondary hero: `Lihat Tiga Template`
- pricing CTA: `Mulai Gratis` / `Pilih Paket Signature`
- closing CTA: `Mulai Rancang Undangan`
