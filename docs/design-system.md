# Design System

## Color Roles
- primary: `#F0C4CB`
- primary-strong: `#C87D87`
- secondary: `#6B7556`
- accent: `#E5BCA9`
- background-soft: `#FBEAD6`
- text-primary: `#231F20`
- text-secondary: `#6F6465`
- border: `rgba(200, 125, 135, 0.18)`
- success: `#4F7B62`
- warning: `#B68448`
- error: `#B55763`

## Typography
- heading: `Cormorant Garamond`
- body: `Plus Jakarta Sans`

## Type Scale
- hero: `text-[3.2rem]` to `text-[6rem]`
- section heading: `text-4xl` to `text-5xl`
- card heading: `text-lg` to `text-2xl`
- body: `text-sm`, `text-base`, `text-lg`

## Spacing
- compact: `px-4 py-3`
- card: `p-5` or `p-6`
- section: `py-16` or `py-24`
- shell: `max-w-7xl`

## Radius
- card: `2rem`
- panel: `2.75rem`
- pill/button: `999px`

## Shadows
- soft: `0 18px 50px rgba(130, 93, 84, 0.08)`
- float: `0 28px 90px rgba(110, 77, 83, 0.12)`
- button: `0 16px 34px rgba(107, 117, 86, 0.22)`

## Buttons
- primary: olive background, white text
- secondary: translucent white, subtle border

## Cards
- `surface-panel`: glassy panel with blur
- `surface-card`: white card with soft border/shadow

## Form Inputs
- rounded 2xl shape
- thin soft border
- large tap target
- neutral white background

## Motion Rules
- opacity + translate only
- <= 1s duration
- no parallax
- respect `prefers-reduced-motion`
